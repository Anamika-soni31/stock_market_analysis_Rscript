# Stock Market Analysis using R

## Project Overview
This project analyzes weekly S&P 500 stock market data using machine learning and statistical modeling techniques in R. The analysis focuses on predicting market direction using Logistic Regression and K-Nearest Neighbors (KNN).

## Objectives
- Explore stock market return patterns
- Build predictive classification models
- Evaluate model performance using confusion matrices and ROC curves
- Compare Logistic Regression and KNN methods

## Tools & Libraries
- R
- ggplot2
- class
- ROSE

## Techniques Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- ROC Curve Analysis
- AUC Evaluation
- Confusion Matrix
- Accuracy, Sensitivity, and Specificity Analysis

## Files Included
- `stock_market_analysis.Rmd`
- `stock_market_analysis.html`
- `weekly.csv`

## Key Findings
- Lag2 was the most statistically significant predictor in logistic regression.
- Logistic Regression achieved the highest prediction accuracy.
- KNN with k = 10 slightly improved AUC performance.
